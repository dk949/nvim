# nvim
neovim config

## How to use
* clone **recursively** to `$XDG_CONFIG_HOME`
* launch nvim
```vim
:PluginInstall
```

## Plugins dependencies
* coc
    * node js
    * npm
    * everything that mentions '(coc)' as a dependency would also require these
* cmake
    * (coc)
* c++
    * ccls
    * (coc)
* c++ linting
    * clang-tidy
    * cppcheck
* haskell
    * HLS
    * (coc)
* hexedit
    * xxd
* json
    * (coc)
* jsts
    * (coc)
* man
    * man
* python
    * (coc)
* ripgrep
    * ripgrep
* rust
    * (coc)
* tex
    * latex
